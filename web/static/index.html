使用方法

任务下发
  • url: /api/start
  • method: POST
  • type: JSON
  • param: task_id, target, vid
  • example: curl http://127.0.0.1:5000/api/start -d '{"task_id": "11111111111", "vid": "vb_xxxx_yyyy", "target": "http://x.com/"}'
获取结果
  • url: /api/result
  • method: POST
  • type: JSON
  • param: task_id
  • example: curl http://127.0.0.1:5000/api/result -d '{"task_id": "11111111111"}'
